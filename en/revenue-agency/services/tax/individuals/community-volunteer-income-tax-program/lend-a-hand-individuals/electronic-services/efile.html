Overview

EFILE is an automated service that allows CVITP volunteers and other approved tax preparation service providers to electronically send individual income tax return information to the Canada Revenue Agency (CRA).

The EFILE web service is available to transmit returns from the 2017 tax year to present.

EFILE transmission services are open for the majority of the year, typically starting in February. Exact dates for the current year can be found in What’s new for this filing season.

Registering to use EFILE

You must only apply for an EFILE number if you intend to file returns electronically. Your EFILE number must be renewed annually.

Volunteers who are approved to file electronically are not required to provide a police records check, unless they are using the Auto-fill my return service.

For more information, refer to Register to volunteer. 

Important note

Although you will be issued an EFILE number and password as soon as you apply, your application may take up to 30 business days to process. To give us enough time to review your application before the filing season starts, we encourage you to file your application before December 31.

Getting ready to use EFILE

If you are working on a computer that is also used by other volunteers, you must open the EFILE-NetFile setup and ensure that your EFILE number, password and COIN (if available) are entered correctly before you complete any tax returns, every time you use the computer.

To set up EFILE, refer to the instructions in EFILE-NetFile setup.

EFILING the tax return

There are a number of steps that the software will take you through in order to successfully transmit a tax return using EFILE. For screen-by-screen instructions, refer to EFILING the return.

In order to complete prior year returns with UFile CVITP software, you must install the applicable version for each year you wish to prepare. If you are preparing prior year tax returns, select Yes in response to the question Will you be using EFILE to submit a prior year federal return, today or within the next two days?

Nice to know

You cannot transmit returns electronically prior to 2017. However, you can still complete and print the returns using UFile CVITP. You can prepare tax returns as far back as ten calendar years. For information on how to print returns, refer to Providing the tax return and forms.

When you prepare previous year returns, it is recommended that you begin with the earliest year and continue in order.

Once you complete all steps to EFILE the tax return and have clicked File my return now!, UFile connects directly to the CRA to transmit the tax return. It may take a few minutes to transmit the return as the system conducts an initial review on each tax return transmitted.

For screen-by-screen instructions on Accepted or Rejected EFILE transmissions, refer to EFILING the tax return.

Verifying EFILE transmission errors

The system can reject an EFILE transmission multiple times with different error codes.

If you receive error messages when you transmit the tax return, refer to the RC4018, Electronic Filers Manual, Chapter 2.

The manual explains how the error codes are set, what they mean, and how to respond to them.

Correct the information as instructed in the manual before you attempt to transmit the return again.

If you cannot find an error code in the RC4018, or you need additional help to resolve an error code, contact your regional EFILE Helpdesk support. This information can also be found under useful links.

When you attempt to re-transmit a return after you correct an error, it is important to ensure that you have received either a green Accepted status, or a red Rejected status in the Federal EFILE Report generated by UFile CVITP before you move on to a new individual's tax return.

If the Federal EFILE Report status for the re-transmission is:

  • Accepted - proceed to the steps found under After preparing the income tax return
  • Rejected - print a paper copy of the return for the taxpayer and instruct them to mail it to the CRA for processing

To ensure you provide the individual with the correct mailing address for their applicable Tax Centre, refer to Find a CRA address - Canada.ca.

Nice to know

Direct deposit enrollment forms are no longer included in the tax packages. However, the option is still available for returns that are filed electronically through EFILE.
