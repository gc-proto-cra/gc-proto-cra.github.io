Directives

Pendant l’essai, le modérateur vous présentera un certain nombre de tâches différentes, une à la fois. Une fois que vous avez lu et compris la tâche, cliquez sur le bouton « Start Task » (Commencer la tâche). Cela ouvrira un site Web dans un nouvel onglet. Utilisez le site Web pour essayer d’effectuer la tâche. Lorsque vous avez terminé, revenez à cet onglet et sélectionnez votre réponse.

N’utilisez pas Google pour trouver les réponses lorsque vous effectuez les tâches. Utilisez le site Web de l’Agence du revenu du Canada pour trouver les réponses aux questions qui vous sont posées. Nous testons les pages Web, et non vos connaissances. Si vous connaissez déjà la réponse, essayez de trouver la réponse sur le site Web.

N’oubliez pas de nous dire ce que vous pensez et de quelle façon vous naviguez alors que vous parcourez les scénarios.
